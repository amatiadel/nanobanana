# Product Overview

## Banana Prompts

A community-driven AI prompt gallery web application that enables users to discover, explore, and share AI-generated image prompts.

### Product Description

Banana Prompts is a modern web application for browsing and discovering AI image prompts. Users can explore a curated collection of prompts with advanced filtering, search capabilities, and detailed prompt information. The application features a clean, modern design with soft backgrounds, rounded cards, and intuitive navigation.

### Target Users

- AI art enthusiasts looking for inspiration
- Content creators seeking quality prompts
- Digital artists exploring AI-generated imagery
- Anyone interested in AI image generation

### Key Features

1. **Browse Gallery**: Infinite scroll grid of prompt cards with images and metadata
2. **Advanced Filtering**: Search by keywords and filter by multiple tag categories
3. **Detailed Views**: Full prompt information with related suggestions
4. **Copy Functionality**: Easy clipboard copy for prompts
5. **Responsive Design**: Optimized for desktop, tablet, and mobile devices
6. **Performance**: Fast loading with image optimization and lazy loading

### Domain Terminology

- **Prompt**: Text instructions used to generate AI images
- **Tag**: Categorized label for classifying prompts (e.g., "cinematic", "realistic")
- **Tag Categories**: Artistic Styles, Corporate & Professional, Genre & Themes, Mood & Tone, Optional Add-ons
- **Premium Prompt**: Featured or high-quality prompts marked with special badge
- **Creator**: User who contributed the prompt
